# Atmos Weather Tracker
### A weather application built with HTML, CSS, and JavaScript that provides real-time weather data using the Open-Meteo API (free, no API key required).

### Features
🔍 City Search with Autocomplete — Instant location suggestions as you type
🌡️ Real-Time Weather Data — Temperature, humidity, wind speed, visibility, and feels-like
🎬 Dynamic Weather Animations — Canvas-based particle effects that change based on current conditions (rain, snow, storms, etc.)
🖥️ Event Loop Console — Educational panel demonstrating sync, async, microtask, and macrotask execution in real-time
📜 Search History — Persisted in LocalStorage with one-click re-search
📱 Responsive Design — Works on desktop and mobile
### Tech Stack
HTML5, CSS3, JavaScript
Open-Meteo API — Geocoding & weather data
Google Fonts — Plus Jakarta Sans & DM Mono
